# Mizuki 配置与扩展体系

本文说明 `config.ts` 的模块划分、国际化、侧边栏与导航配置，以及如何添加新页面和新功能。

---

## 一、config.ts 模块总览

`src/config.ts` 导出多个配置对象，由各组件按需引用：

| 导出名 | 用途 |
|--------|------|
| `siteConfig` | 站点标题、URL、语言、主题色等 |
| `navBarConfig` | 顶栏导航链接 |
| `profileConfig` | 侧边栏头像、昵称、社交链接 |
| `sidebarLayoutConfig` | 侧边栏组件及布局 |
| `announcementConfig` | 公告栏（可关闭） |
| `musicPlayerConfig` | 音乐播放器 |
| `banner` | 横幅图片与效果 |
| `footerConfig` | 页脚自定义 HTML |
| 等 | 见文件内注释 |

## 二、导航配置（navBarConfig）

### 2.1 使用预设链接

`LinkPreset` 是预定义的导航项，会自动使用 i18n 文案：

```ts
import { LinkPreset } from "./types/config";

navBarConfig: {
  links: [
    LinkPreset.Home,    // 首页，文案来自 i18n
    LinkPreset.Archive,
    LinkPreset.Diary,
    LinkPreset.About,
  ],
}
```

### 2.2 自定义链接

```ts
{
  name: "编程与技术",
  url: "/tech/",
  icon: "material-symbols:code",
}
```

- `name`：显示文字
- `url`：站内路径，建议以 `/` 结尾
- `icon`：Iconify 图标，格式 `集合:图标名`，如 `material-symbols:code`

### 2.3 完整示例

```ts
export const navBarConfig: NavBarConfig = {
  links: [
    LinkPreset.Home,
    LinkPreset.Archive,
    { name: "编程与技术", url: "/tech/", icon: "material-symbols:code" },
    { name: "数学与理科", url: "/science/", icon: "material-symbols:functions" },
    LinkPreset.Diary,
    { name: "关于我", url: "/about/", icon: "material-symbols:info" },
  ],
};
```

## 三、侧边栏配置（sidebarLayoutConfig）

### 3.1 组件列表（properties）

```ts
properties: [
  { type: "profile", position: "top", animationDelay: 0 },
  { type: "categories", position: "sticky", animationDelay: 150 },
  { type: "tags", position: "top", animationDelay: 250 },
  { type: "site-stats", position: "top", animationDelay: 200 },
  { type: "calendar", position: "top", animationDelay: 250 },
]
```

- `type`：组件类型，如 `profile`、`categories`、`tags`、`music-player` 等
- `position`：`top` 固定顶部，`sticky` 可随滚动
- `animationDelay`：入场动画延迟（毫秒）

### 3.2 布局分配（components）

```ts
components: {
  left: ["profile", "categories", "tags"],      // 左侧栏
  right: ["site-stats", "calendar"],           // 右侧栏
  drawer: ["profile", "categories", "tags"],   // 移动端抽屉
}
```

- `left`、`right` 对应桌面端左右侧栏
- `drawer` 为移动端侧滑抽屉中的组件
- 需在 `layout.position` 为 `both` 时才有右侧栏

### 3.3 添加公告组件

在 `left` 和 `drawer` 中加入 `announcement`：

```ts
left: ["profile", "announcement", "categories", "tags"],
drawer: ["profile", "announcement", "categories", "tags"],
```

并在 `properties` 中增加：

```ts
{ type: "announcement", position: "top", animationDelay: 50 },
```

## 四、国际化（i18n）

### 4.1 语言文件位置

```
src/i18n/
├── i18nKey.ts        # 文案 Key 枚举
├── translation.ts    # 根据 lang 选择语言
└── languages/
    ├── zh_CN.ts      # 简体中文
    ├── zh_TW.ts      # 繁体中文
    ├── en.ts         # 英文
    └── ja.ts         # 日文
```

### 4.2 添加新文案

**步骤 1**：在 `i18nKey.ts` 中增加 Key

```ts
export enum Key {
  // ...
  myNewText = "myNewText",
}
```

**步骤 2**：在各语言文件中添加翻译

```ts
// zh_CN.ts
[Key.myNewText]: "我的新文案",

// en.ts
[Key.myNewText]: "My new text",
```

**步骤 3**：在组件中使用

```ts
import { i18n } from "../i18n/translation";
import Key from "../i18n/i18nKey";

// 使用
{i18n(Key.myNewText)}
```

### 4.3 站点语言

`config.ts` 中：

```ts
const SITE_LANG = "zh_CN";
```

修改后，全站默认使用对应语言包。

## 五、数据源配置

### 5.1 友链（friends.ts）

```ts
// src/data/friends.ts
export const friends: Friend[] = [
  {
    name: "友链名称",
    url: "https://example.com",
    imgurl: "https://example.com/avatar.png",
    siteurl: "https://example.com",
    description: "描述",
  },
];
```

### 5.2 时间线（timeline.ts）

```ts
// src/data/timeline.ts
{
  year: "2026",
  items: [
    {
      date: "2026-03-04",
      title: "事件标题",
      description: "描述",
      url: "https://...",
      type: "website",  // website | certificate | project | other
    },
  ],
}
```

### 5.3 技能（skills.ts）

按分类组织技能项，用于技能页展示。

### 5.4 项目（projects.ts）

项目列表，含名称、描述、链接、技术栈等。

### 5.5 日记（diary.ts）

日记条目，支持时间、内容、图片、心情、位置等。

## 六、添加新页面

### 6.1 新建页面文件

在 `src/pages/` 下新建 `my-page.astro`：

```astro
---
import MainGridLayout from "../layouts/MainGridLayout.astro";
---

<MainGridLayout title="我的页面" description="页面描述">
  <p>页面内容</p>
</MainGridLayout>
```

构建后访问 `/my-page/`。

### 6.2 加入导航

在 `navBarConfig.links` 中增加：

```ts
{ name: "我的页面", url: "/my-page/", icon: "material-symbols:star" },
```

### 6.3 可选：加入 featurePages

若需像「番剧」「友链」一样可开关，在 `siteConfig.featurePages` 中增加对应字段，并在相关页面中判断 `siteConfig.featurePages.xxx` 决定是否渲染。

---

## 七、添加新侧边栏组件

### 7.1 创建组件

在 `src/components/widget/` 下新建 `MyWidget.astro` 或 `MyWidget.svelte`。

### 7.2 注册组件

在 `utils/widget-manager.ts` 中：

```ts
const widgetMap = {
  // ...
  "my-widget": "../components/widget/MyWidget.astro",
};
```

### 7.3 在 types 中增加类型

在 `types/config.ts` 的 `WidgetComponentType` 中加入 `"my-widget"`。

### 7.4 加入侧边栏配置

在 `sidebarLayoutConfig` 的 `properties` 和 `components.left/right/drawer` 中加入 `"my-widget"`。

## 八、常见修改示例

### 8.1 修改站点标题

```ts
siteConfig: {
  title: "我的自留地",
  subtitle: "记录编程、数学、生活的点滴",
}
```

### 8.2 修改头像和昵称

```ts
profileConfig: {
  avatar: "assets/images/avatar.jpg",
  name: "你的昵称",
}
```

### 8.3 添加社交链接

```ts
profileConfig: {
  links: [
    { name: "Bilibili", icon: "fa7-brands:bilibili", url: "https://..." },
    { name: "GitHub", icon: "fa7-brands:github", url: "https://..." },
  ],
}
```

### 8.4 关闭某功能

```ts
featurePages: {
  anime: false,   // 关闭番剧页
  albums: false,  // 关闭相册页
}

musicPlayerConfig: {
  enable: false,  // 关闭音乐播放器
}
```

关闭 `featurePages` 中的页面后，记得从 `navBarConfig.links` 中移除对应链接。
