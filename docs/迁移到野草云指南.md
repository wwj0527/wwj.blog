# 博客迁移到野草云香港服务器指南

从腾讯云迁移到野草云香港 VPS，按步骤执行即可。

---

## 前置准备

- 野草云香港服务器已购买，获得 **公网 IP** 和 **SSH 登录信息**
- 本地能通过 SSH 连接服务器

---

## 第一步：SSH 连接服务器

```bash
ssh root@你的野草云IP
```

输入密码登录。若使用密钥，则：

```bash
ssh -i 你的密钥路径 root@你的野草云IP
```

---

## 第二步：安装基础环境

### 2.1 安装 Node.js 和 pnpm

```bash
# 安装 Node.js 20（以 Ubuntu/Debian 为例）
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt-get install -y nodejs

# 安装 pnpm
npm install -g pnpm

# 验证
node -v   # 应显示 v20.x.x
pnpm -v
```

### 2.2 安装 Git

```bash
apt-get update
apt-get install -y git
```

### 2.3 安装 Nginx

```bash
apt-get install -y nginx
systemctl enable nginx
systemctl start nginx
```

---

## 第三步：克隆项目并构建

```bash
# 创建网站目录
mkdir -p /www/wwwroot/20040527.xyz
cd /www/wwwroot/20040527.xyz

# 克隆仓库（用 Gitee，国内拉取更快；或 GitHub）
git clone https://gitee.com/wwj040527/wwj.blog.git .

# 或从 GitHub
# git clone https://github.com/wwj0527/wwj.blog.git .

# 安装依赖并构建
pnpm install
pnpm build
```

构建完成后，静态文件在 `dist` 目录。

---

## 第四步：配置 Nginx

创建站点配置：

```bash
nano /etc/nginx/sites-available/20040527.xyz
```

写入以下内容（把 `你的野草云IP` 换成实际 IP，或先留空，等 DNS 生效后再改）：

```nginx
server {
    listen 80;
    server_name 20040527.xyz www.20040527.xyz;
    root /www/wwwroot/20040527.xyz/dist;
    index index.html;

    location / {
        try_files $uri $uri/ $uri.html /index.html;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
        expires 7d;
        add_header Cache-Control "public, immutable";
    }
}
```

启用配置并重载 Nginx：

```bash
ln -sf /etc/nginx/sites-available/20040527.xyz /etc/nginx/sites-enabled/
nginx -t
systemctl reload nginx
```

---

## 第五步：配置 DNS 解析

1. 登录 [Spaceship](https://www.spaceship.com) → 进入 `20040527.xyz` 的 DNS 管理
2. 添加/修改 A 记录：
   - **@** → 野草云服务器 IP
   - **www** → 野草云服务器 IP
3. 删除或修改旧的指向腾讯云的 A 记录

DNS 生效通常需要几分钟到几小时。

---

## 第六步：验证

1. **用 IP 直接访问**：`http://你的野草云IP`（若 Nginx 默认站指向 dist，可先这样测）
2. **用域名访问**：`https://20040527.xyz`（DNS 生效后）

若能看到博客首页，说明迁移成功。

---

## 第七步：配置 HTTPS（可选但推荐）

使用 Let's Encrypt 免费证书：

```bash
apt-get install -y certbot python3-certbot-nginx
certbot --nginx -d 20040527.xyz -d www.20040527.xyz
```

按提示输入邮箱、同意条款，certbot 会自动配置 HTTPS。

## 常见问题

**Q：访问显示 502？**  
检查 `dist` 是否存在：`ls /www/wwwroot/20040527.xyz/dist`

**Q：克隆很慢？**  
可先用 GitHub，或配置 Git 代理；野草云香港一般拉 GitHub 会快一些。

**Q：需要配置防火墙？**  
确保放行 80、443 端口。野草云控制台一般有安全组/防火墙设置。
